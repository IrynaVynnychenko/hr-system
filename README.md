# HR System - Portfolio Project

This project contains a full-featured HR system and promotional pages built using components, Gulp, SCSS, and Pug.

## 🚀 Project Setup

To run the local server, use:
```bash
# Install dependencies
npm install

# Run build
npm run build

# Start local server
npm start
```
